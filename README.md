# README

## Prérequis
- Node.js (recommandé : version LTS)
- npm (ou yarn / pnpm)

## Installation
```bash
npm install
```

## Lancer en développement
```bash
npm run dev
```

## Build (production)
```bash
npm run build
```

## Prévisualiser le build
```bash
npm run preview
```

## Tech
- Vite + React
- React Router
- i18next (traductions)
- country-flag-icons
